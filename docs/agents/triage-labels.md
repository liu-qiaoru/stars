# Triage labels

工程技能使用以下五种标准状态。本地 Markdown tracker 通过 issue 文件中的 `Status:` 字段保存状态。

| 标准角色 | 本仓库状态 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 等待维护者评估 |
| `needs-info` | `needs-info` | 等待补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整，Agent 无需额外上下文即可执行 |
| `ready-for-human` | `ready-for-human` | 需要人工实施 |
| `wontfix` | `wontfix` | 明确不实施 |

技能提到某个标准角色时，使用本表中对应的状态名称，不创建同义状态。
